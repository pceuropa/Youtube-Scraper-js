# Youtube-Scraper-js
Scraper written in JavaScript to get some information from Youtube Video page.

## Structure
| file  | description  |
|---|---|
| cralwer.js | main file |
| ./inotifywait.sh | Watch and run file |

## requirements
* NodeJs
* JavaScript

## Instalation
```
npm install
```

node cralwer.js
